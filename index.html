Array Operations

This is a demonstration of various array operations in JavaScript.

Array Basics

  • Array: A collection of elements with different data types (except in C and C++).
  • Push: Insert an element at the end of the array.
  • Pop: Remove the last element from the array.
  • Shift: Remove the first element from the array.
  • Unshift: Insert an element at the beginning of the array.

More Methods

  • indexOf: Find the index of an element.
  • splice: Add or remove elements from an array at a specific index.
  • includes: Check if an element is present in the array.
  • join: Convert the array to a string with elements joined by a specified separator.
  • Array.isArray: Check if a variable is an array.
  • reverse: Reverse the order of elements in the array.
  • sort: Sort the elements of the array.
